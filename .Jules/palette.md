# Palette's Journal

## 2025-12-23 - CLI Visual Polish
**Learning:** In scientific/data CLI tools, users often run scripts to verify outputs. Providing a side-by-side comparison (Start vs Output vs End) instead of just the output significantly reduces the cognitive load of verification.
**Action:** When creating visualization scripts for interpolation or transformation tasks, always default to generating a comparative view (e.g., Triptych) rather than a single artifact.
